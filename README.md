Project Title: Car Price Prediction & Data Analysis
This repository contains a mini-project focused on data analysis, visualization, and machine learning for predicting car prices.

Project Overview
The goal of this project is to analyze automotive data to understand the factors influencing car prices and to build a predictive model. The project leverages Excel for initial data handling and Power BI for visualizing insights.
